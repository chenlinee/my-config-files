## my linux rc file

## install

``` bash
git clone https://github.com/chenlinee/my-config-files.git
cd my-config-files
bash auto_script/init.sh

```

## install nvim/vim plug

  * open nvim/vim
  * install nvim/vim plug: `:PlugInstall`
  * install coc.nvim plug
    * `:CocInstall coc-clangd`
    * `:CocInstall coc-json`
    * `:CocInstall coc-pyright`
